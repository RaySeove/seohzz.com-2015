系统环境：Ubuntu

1、系统更新

apt-get update

apt-get upgrade

2、nginx

安装：apt-get install nginx

启动：/etc/init.d/nginx start

停止：service nginx stop

重启：service nginx restart

3、git

安装：apt-get install git

4、安装rvm（用于管理ruby）及最新版ruby

\curl -L https://get.rvm.io | bash -s stable --ruby

启动：source /usr/local/rvm/scripts/rvm

检查安装：rvm -v

查看所有ruby版本：rvm list known

安装2.0.0[-p481]：rvm install 2.0.0-p481

设置ruby默认版本：rvm --default use 2.0.0-p481

5、Jekyll及相关软件

git clone git://github.com/qnn/qnnsafe.com-v1.git

cd qnnsafe.com-v1

bundle

gem install -V jekyll ruby-nuggets rdiscount coffee-script therubyracer uglifier sass jekyll-assets haml unicode_utils html2haml rubygems

gem install rails

6、安装zip及使用

安装：apt-get install zip

压缩当前文件夹：zip -r ./压缩包名称.zip ./*

解压：unzip 压缩包名称.zip
